# 🕵️ IP Detective - Công Cụ Tra Cứu Vị Trí IP

Ứng dụng web tra cứu thông tin vị trí địa lý của bất kỳ địa chỉ IP, IPv6 hoặc tên miền nào.

## Tính năng
- 🌍 Tự động phát hiện IP của bạn khi mở
- 🗺️ Hiển thị bản đồ Google Maps & OpenStreetMap
- 🔒 Phát hiện Proxy/VPN, Mobile, Hosting
- 📋 Chế độ Batch: tra nhiều IP cùng lúc, xuất CSV
- 🕒 Lịch sử 10 tra cứu gần nhất
- 🌙 Giao diện sáng/tối

## Chạy ứng dụng

```bash
pip install -r requirements.txt
streamlit run app.py
```
